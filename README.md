# cruise-romantic-getaway · Claude Code 插件

为情侣提供浪漫邮轮之旅的精选产品，打造难忘的回忆。

![published with craftwave](https://img.shields.io/badge/published%20with-craftwave-6366f1)

## 安装

在 Claude Code 中执行：

```
/plugin marketplace add OWNER/cruise-romantic-getaway-claude-plugin
/plugin install cruise-romantic-getaway@cruise-romantic-getaway-marketplace
```

> 将 `OWNER` 替换为本仓库所属的 GitHub 账号/组织名。

安装后即可在 Claude Code 里使用技能 **cruise-romantic-getaway**。本仓库由
[CruiseSkillBridge](https://cruiseskillbridge.com) 按 Claude Code 插件市场规范自动生成。
